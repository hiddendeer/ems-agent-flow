## 🛡️ 安全机制

### 三重审查机制
1. **参数验证**: 确保参数合法、在允许范围内
2. **安全检查**: 检查设备状态、时序冲突、安全阈值
3. **权限验证**: 验证操作者是否有权限执行该操作

### 安全阈值
- **功率范围**: 0-5000kW（不超过额定功率110%）
- **SOC范围**: 10%-95%（极端情况5%-98%）
- **温度范围**: -10℃至55℃（>50℃禁止大功率，>55℃禁止操作）
- **电压范围**: 300V-450V（建议350-420V）
- **电流范围**: -1000A至1000A

### 风险等级
- **低风险**: 常规操作，可直接执行
- **中风险**: 需要关注，建议执行
- **高风险**: 建议二次确认
- **严重风险**: 必须明确确认，否则拒绝

## 📝 使用示例

### 示例 1: 正常充电指令

```python
from agents.domains.command_execution import CommandExecutionAgent

agent = CommandExecutionAgent()
tools = {t.name: t for t in agent.get_tools()}

# 审查指令
review_result = tools["review_and_validate_command"].invoke({
    "command_type": "charge",
    "device_id": "BAT-001",
    "parameters": {"power": 100, "target_soc": 80},
    "operator": "user_001"
})
print(review_result)

# 评估风险
risk_result = tools["assess_command_risk"].invoke({
    "command_type": "charge",
    "device_id": "BAT-001",
    "parameters": {"power": 100, "target_soc": 80}
})
print(risk_result)

# 执行指令
exec_result = tools["execute_device_command"].invoke({
    "command_type": "charge",
    "device_id": "BAT-001",
    "parameters": {"power": 100, "target_soc": 80},
    "operator": "user_001"
})
print(exec_result)
```

### 示例 2: 查询设备状态

```python
status = tools["query_device_status"].invoke({
    "device_id": "BAT-001"
})
print(status)
```

### 示例 3: 紧急停止

```python
emergency = tools["emergency_stop_all"].invoke({
    "scope": "all",
    "reason": "检测到严重安全风险"
})
print(emergency)
```
