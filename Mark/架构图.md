## 数据流

```mermaid
graph LR
    User[👤 用户] --> EMS[create_ems_agent]
    EMS --> DA[deepagents<br/>create_deep_agent]
    
    Registry[AgentRegistry] --> |subagent configs| DA
    
    DA --> |编译| LG[LangGraph<br/>CompiledStateGraph]
    
    LG --> |task 工具| ES[EnergyStorageExpert<br/>子图]
    LG --> |task 工具| PM[PowerMarketExpert<br/>子图]
    LG --> |task 工具| GP[general-purpose<br/>通用子图]
    
    ES --> |tool call| EST[电池状态/PCS/套利计算]
    PM --> |tool call| PMT[电价查询/政策/负荷]
```

## 使用示例

```python
from src.agents import create_ems_agent, register_all_domains

# 1. 注册所有领域 Agent
register_all_domains()

# 2. 创建 EMS Lead Agent（deepagents CompiledStateGraph）
agent = create_ems_agent()

# 3. 执行 — deepagents 自动通过 task 工具调度领域 SubAgent
result = await agent.ainvoke({
    "messages": [("user", "调研广东省电价并制定储能充放电策略")]
})

# 4. 流式执行
async for step in agent.astream({
    "messages": [("user", "查看电池SOC状态")]
}):
    print(step)
```
