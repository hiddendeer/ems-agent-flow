# 电力市场现货交易申报分析规约 (SOP)

## 1. 概述
在分析和计算任何电力现货交易、日前申报等相关任务时，必须严格参考本手册。该技能专门用于在储能削峰填谷以及集中式新能源入市场景中指引系统计算。

本规范适用于各类电力现货市场环境，提供通用的交易申报分析框架和计算方法。具体市场参数应根据所在地区的市场规则进行配置。

## 2. 核心指标限制

### 2.1 价格限制
- **出清价格上下限**: 根据各地市场政策，现货市场设有价格限价范围：
  - 最高限价：通常不超过 1.5 元/度（具体以当地市场规则为准）
  - 最低限价：不得低于 0 元/度或规定的负值下限
  - 价格申报应在限价范围内进行，避免超出市场规则的报价

### 2.2 功率与爬坡限制
- **爬坡率限制**: 当储能系统（或发电机组）在时段切换时，出力调整幅度限制：
  - 储能系统：一小时内功率变化不可超过其最大容量的 40%-100%（根据电池特性确定）
  - 火电机组：一小时内爬坡能力通常不超过额定容量的 30%-50%
  - 新能源机组：需考虑预测误差和调节能力

### 2.3 容量与电量限制
- **额定容量**: 设备的额定功率输出能力（MW）
- **可用容量**: 考虑检修、故障等因素后的实际可用功率
- **电量约束**: 储能系统的能量容量（MWh），需考虑荷电状态（SOC）限制

## 3. 标准化工作流

当你接到关于"制定现货交易策略"或"计算竞价收益"指令时，务必按以下步骤执行：

### 3.1 前置数据准备
1. **调用价格预测工具**: 唤起 `query_electricity_price` 工具获取当下或预测的电价数据
2. **获取设备状态**: 查询当前设备的可用容量、SOC水平、运行约束
3. **收集市场信息**: 了解当前市场的供需情况、网络约束、价格走势

### 3.2 边界条件校验
1. **价格合理性检查**: 如果预测某一时段的电价低于边际成本或处于不合理区间：
   - 设定为不申报时段
   - 或申报极高价避免被调用（策略性报价）
2. **功率可行性验证**: 确保申报功率在设备技术约束范围内
3. **爬坡能力校核**: 验证相邻时段之间的功率变化满足爬坡率要求
4. **能量平衡检查**: 对于储能系统，确保充放电能量在整个周期内平衡

### 3.3 策略优化计算
1. **目标函数**: 根据运营目标设定优化目标（收益最大、成本最低、风险最小等）
2. **约束条件**: 考虑设备约束、市场规则、网络约束等
3. **优化算法**: 采用适当的优化方法求解最优申报策略

### 3.4 结果输出标准
最终结果输出给主控 Agent 时，必须包含：
1. **申报时段**: 明确标注参与申报的时间段
2. **申报功率**: 各时段的申报功率值（MW）
3. **申报价格**: 各时段的申报价格（元/MWh）
4. **预期收益**: 根据预测计算的预期收益
5. **风险评估**: 策略的风险分析和应对措施
6. **技能引用**: 必须明确标注【技能引用：现货交易申报规范 V1.0】

## 4. 常见应用场景

### 4.1 储能削峰填谷
- **低价时段充电**: 在电价较低的时段（通常为夜间）购买电力充电
- **高价时段放电**: 在电价较高的时段（通常为白天高峰）放电获利
- **SOC管理**: 合理管理荷电状态，避免深度充放电影响电池寿命
- **辅助服务**: 参与调频、备用等辅助服务市场获得额外收益

### 4.2 新能源入市
- **预测偏差管理**: 考虑新能源出力预测的不确定性，制定灵活申报策略
- **弃风弃光减少**: 通过市场申报减少弃风弃光损失
- **曲线优化**: 根据市场价格信号优化出力曲线
- **风险管理**: 利用金融工具对冲价格和出力风险

### 4.3 需求响应
- **负荷转移**: 将用电负荷从高电价时段转移到低电价时段
- **负荷削减**: 在高价时段减少用电，获得补偿
- **灵活性资源**: 将可调节负荷作为灵活性资源参与市场

## 5. 计算方法与公式

### 5.1 收益计算
总收益 = Σ(时段电价 × 该时段电量) - 运营成本 - 辅助服务成本

### 5.2 边际成本计算
边际成本 = 增量成本 / 增量发电量

### 5.3 SOC计算（储能系统）
SOC(t) = SOC(t-1) + (充电功率 - 放电功率) × Δt / 电池容量

## 6. 风险控制要求

### 6.1 价格风险
- 设置价格上下限，避免极端价格损失
- 建立价格预测模型，实时跟踪市场价格走势
- 制定应急预案，应对价格剧烈波动

### 6.2 运行风险
- 确保设备运行在安全范围内
- 建立设备故障应急预案
- 定期校验预测精度和策略效果

### 6.3 合规风险
- 严格遵守市场规则和监管要求
- 避免市场操纵和不当报价行为
- 建立合规审查机制

## 7. 参数配置说明

实际应用时，需要根据具体市场规则配置以下参数：
- 市场价格上下限
- 最小申报单位
- 爬坡率限制
- 功率调节范围
- 时长粒度（15分钟、30分钟、1小时等）
- 结算规则
