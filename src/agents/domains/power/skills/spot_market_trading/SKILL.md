# 电力市场现货交易申报分析规约 (SOP)

## 1. 概述
在分析和计算任何电力现货交易、日前申报等相关任务时，必须严格参考本手册。该技能专门用于在储能削峰填谷以及集中式新能源入市场景中指引系统计算。

## 2. 核心指标限制
- **出清价格上下限**: 根据政策，现货市场最高限价通常不可超过 `1.5 元/度`，最低限价不得低于 `0 元/度`。
- **爬坡率限制**: 当储能系统（或火电机组）在时段切换时，出力调整幅度在一小时内不可超过其最大容量的 `40%`。

## 3. 标准化工作流
当你接到关于“制定现货交易策略”或“计算竞价收益”指令时，务必分三步执行：
1. **调用前置工具**：先唤起 `query_electricity_price` 工具获取当下或模拟的电价。
2. **应用边界卡点**：如果预测市场的某一时段出现亏损风险（电价跌破边际成本），应当自动设定该时段不申报或申报极高价避免被调用。
3. **输出标准制式**：最终结果输出给主控 Agent 时，必须明确标注【技能引用：现货交易申报规范 V1.0】。
