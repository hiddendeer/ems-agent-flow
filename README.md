## 🎖️ 这是什么？

想象一下，如果**储能电站、电力交易、政策分析**这些复杂工作，都有一个 AI 团队在帮你处理，会是什么感觉？

**EMS Agent Flow** 就是这个 AI 团队。它不是一个简单的软件，而是：

- 🏢 **懂储能**的运营专家
- 📊 **懂市场**的交易员
- 🔍 **懂政策**的分析师
- ⚡ **懂设备**的工程师

**5 位 AI 专家** 24/7 为你工作，帮你做出更明智的能源决策。

## 🏗️ 系统是怎么工作的？

### 简单来说：

```
你的问题
    ↓
【首席协调官】理解并分解任务
    ↓
┌──────────┬──────────┬──────────┬──────────┬──────────┐
│储能专家  │市场专家  │搜索专家  │建模专家  │执行专家  │
│          │          │          │          │          │
│计算最优  │分析电价  │搜索政策  │仿真验证  │安全执行  │
│充放电    │预测趋势  │提炼要点  │评估收益  │操作日志  │
└──────────┴──────────┴──────────┴──────────┴──────────┘
    ↓
【整合答案】给你清晰的建议
```

### 核心优势：

| 特性 | 说明 |
|------|------|
| ⚡ **极速响应** | 13秒给出答案，不等茶凉 |
| 🔒 **安全第一** | 每个操作都经过多层安全检查 |
| 🧠 **越用越懂你** | 记住你的偏好，下次更精准 |
| 📱 **随时可用** | 24/7 在线，从不休息 |

## 🚀 如何开始使用？

### 第一步：安装 (2分钟)

```bash
# 1. 下载项目
git clone https://github.com/hiddendeer/ems-agent-flow.git
cd ems-agent-flow

# 2. 安装依赖
pip install -r requirements.txt
```

### 第二步：配置 (1分钟)

```bash
# 复制配置文件
cp .env.example .env

# 编辑 .env，填入你的 API 密钥
# 必填项：
# - OPENAI_API_KEY=你的密钥
# - TAVILY_API_KEY=搜索服务密钥
```

### 第三步：启动 (10秒)

```bash
# 启动服务
python -m src.agents.demo.multi_agent_demo
```

### 第四步：开始提问！

```
你: 帮我分析江苏峰谷电价政策，制定储能充放电策略

AI (思考中...)
✅ 已获取江苏最新电价数据
✅ 峰段: 1.05元/kWh，谷段: 0.28元/kWh
✅ 最优策略: 谷段充电，峰段放电
✅ 预计年收益: 245,280元
✅ 投资回收期: 3.2年
```

**就这么简单！** 🎉

## 💻 技术架构（给开发者）

如果你是开发者，想知道技术细节：

### 核心技术栈

| 组件 | 技术 | 用途 |
|------|------|------|
| AI 大脑 | GLM-4 / GPT-4 | 理解问题、生成答案 |
| 团队协作 | deepagents | 让多个 AI 专家协同工作 |
| Web 服务 | FastAPI | 提供 API 接口 |
| 实时搜索 | Tavily AI | 搜索最新政策和新闻 |
| 优化计算 | PyPSA | 计算最优调度方案 |
| 数据存储 | InfluxDB | 存储设备运行数据 |

### 项目结构

```
ems-agent-flow/
├── src/
│   ├── agents/           # AI 专家团队
│   │   ├── core/        # 核心框架
│   │   └── domains/     # 各领域专家
│   │       ├── energy_storage/    # 储能专家
│   │       ├── power/             # 电力市场专家
│   │       ├── search/            # 搜索专家
│   │       ├── pypsa_modeling/    # 建模专家
│   │       └── command_execution/ # 执行专家
│   ├── mcp/              # 对接 Claude Desktop
│   └── main.py           # Web 服务入口
├── workspaces/           # 用户数据和记忆
└── docs/                 # 文档
```

## 🔧 常见问题

### Q1: 需要什么设备？

**A:** 只需要一台普通电脑即可：
- CPU: 双核及以上
- 内存: 4GB 及以上
- 系统: Windows / macOS / Linux
- 网络: 需要互联网连接（用于搜索服务）

### Q2: API 密钥怎么获取？

**A:**
1. **OpenAI API**: 访问 https://platform.openai.com
2. **Tavily API**: 访问 https://tavily.com
3. 都有免费额度可以试用

### Q3: 数据安全吗？

**A:** 是的，非常安全：
- ✅ 所有数据存储在本地
- ✅ 支持 SQLite 和 MySQL
- ✅ 敏感信息加密存储
- ✅ 完整的操作审计日志

### Q4: 可以自定义吗？

**A:** 完全可以！
- ✅ 可以添加新的 AI 专家
- ✅ 可以对接自己的设备
- ✅ 可以修改决策逻辑
- ✅ 详细的开发文档

### Q5: 支持哪些设备？

**A:** 支持主流储能设备：
- BMS (电池管理系统)
- PCS (储能变流器)
- 电表
- 通过 Modbus、IEC 104 等协议通信
