# Agents 模块说明

## 系统信息
- **操作系统**: Windows 10
- **Python 版本**: 3.11.9

## 模块结构

### config/ 配置模块
负责系统的配置管理，包括环境变量、系统配置等。

### core/ 核心模块
包含系统的核心功能实现，是整个 agent 系统的核心组件。

### demo/ 演示模块
提供示例代码和演示功能，帮助用户了解如何使用 agent 系统。

### tools/ 工具模块
提供各种工具函数和实用程序，支持 agent 的各项功能实现。

## 模块职责

### config 模块
- 系统配置管理
- 环境变量处理
- 配置文件读取和解析

### core 模块
- 核心业务逻辑实现
- Agent 基础功能
- 系统核心算法

### demo 模块
- 示例代码展示
- 功能演示
- 使用教程

### tools 模块
- 通用工具函数
- 辅助功能实现
- 工具类和方法
